# 模型費用與效果比較

[[AI 101 - 主頁|← 回主頁]]

> [!info]
> 整理各家主力模型的 API 定價與 benchmark 表現，幫助你挑到 CP 值最高的模型。
> 資料截至 **2026 年 4 月**，AI 模型迭代極快，使用前建議再確認官方定價頁。

> [!warning] 價格變動頻繁
> LLM API 定價從 2024 到 2026 整體下降約 80%，各家仍持續調整。
> 本文數字僅供參考，以各家官方文件為準。

## API 定價比較（每百萬 tokens）

> **MTok** = 1 million tokens；**1 MTok 中文 ≈ 50–80 萬字**

### Anthropic（Claude）

| 模型 | Input | Output | 特色 |
|---|---|---|---|
| **Haiku 4.5** | $0.80 | $4 | 最快、最省，適合高頻輕量任務 |
| **Sonnet 4.6** | $3 | $15 | 日常開發首選，CP 值最高 |
| **Opus 4.7** | $5 | $25 | 最強推理，但 tokenizer 更新導致實際費用可能高 1–1.35x |

> [!warning] Opus 4.7 的隱藏成本
> 官方定價與 Opus 4.6 相同，但新 tokenizer 對同一段內容可能產生 **1.0–1.35 倍**的 tokens。
> 加上 Extended Thinking 的思考 tokens 另計費，實際帳單可能遠超預期。

## Benchmark 效果比較

> **說明**
> - **SWE-bench** — 程式碼修 bug 能力（越高越強）
> - **MMLU-Pro** — 知識廣度、推理能力
> - **GPQA Diamond** — 研究所級科學推理
> - **Arena Elo** — Chatbot Arena 人類偏好投票

| 模型 | SWE-bench | MMLU-Pro | GPQA Diamond | 強項 |
|---|---|---|---|---|
| **Claude Opus 4.7** | **72.5%** | — | — | 程式碼、複雜推理 |
| **Gemini 3.1 Pro** | 65% | — | **94.3%** | 科學推理、多模態、超長 context |
| **GPT-5.4** | 68% | — | — | 速度、通用均衡 |
| **DeepSeek V3.2** | 67.8% | **85.0%** | 79.9% | 低成本旗艦，AIME 89.3% |
| **Grok 4** | — | — | — | 硬推理 benchmark 第一 |

> [!tip] 沒有單一「最強」模型
> Claude 寫程式最強、Gemini 科學推理最強、DeepSeek 最省錢。
> 真正的問題是：**你的任務是什麼，你的預算是多少？**

## 依任務選模型

| 任務 | 推薦模型 | 理由 |
|---|---|---|
| **日常 coding、筆記、摘要** | Claude Sonnet 4.6 | 均衡，Claude Code 整合最順 |
| **複雜架構設計、難題分析** | Claude Opus 4.7 | SWE-bench 最高，推理最強 |
| **高頻 API 呼叫（自動化）** | Gemini 2.5 Flash / Haiku 4.5 | 超低單價，速度快 |
| **科學研究、長文分析** | Gemini 3.1 Pro | GPQA 最強、1M context |
| **省錢但要旗艦效果** | DeepSeek V3.2 | 同水準下省 10x 成本 |
| **數學 / 競賽推理** | DeepSeek V3.2 Thinking | AIME 89.3%，思考模式 |
| **硬推理挑戰** | Grok 4 | 推理 benchmark 領先 |

## 成本試算範例

以「每天用 AI 寫 1000 字文章」為例（約 2000 tokens in + 1500 tokens out）：

| 模型 | 每天 | 每月（30 天）|
|---|---|---|
| Gemini 2.5 Flash-Lite | $0.0008 | $0.024（幾乎免費）|
| DeepSeek V3.2 | $0.0009 | $0.027 |
| Claude Sonnet 4.6 | $0.0285 | $0.86 |
| Claude Opus 4.7 | $0.0475 | $1.43 |
| GPT-5.4 Pro | $0.294 | $8.82 |

> [!tip] 訂閱 vs API
> 每天只用幾次 → **訂閱方案**（$20/月包含大量用量，CP 值更高）
> 自動化、大量呼叫 → **API**（按量計費，便宜模型省很多）

## Context Window 比較

| 模型 | Context Window |
|---|---|
| Gemini 3.1 Pro | **1,000,000 tokens** |
| Claude Opus 4.7 / Sonnet 4.6 | 200,000 tokens |
| GPT-5.4 | 128,000 tokens |
| DeepSeek V3.2 | 128,000 tokens |

> [!info] Context 越大不代表越好用
> Gemini 的 1M context 適合分析超長文件、整個 codebase。
> 但 context 越大，費用也越高、速度越慢。
> 大多數任務 128K 已足夠。
